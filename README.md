# Akademik-Ajanda
Özellikler:
PHP-PDO ve MySQL kullanılarak geliştirilmiş, son teslim tarihi ve önem derecesi takibi yapılabilen akademik görev yönetim sistemi.

Gecikmiş ödevlerin kırmızı uyarı vermesi.

Önem derecesine göre renkli etiketleme.

Tek tıkla "Tamamla" butonu ve butonun kaybolması.

Nasıl Çalıştırılır?: "SQL dosyasını içeri aktarın, baglan.php ayarlarını yapın ve XAMPP ile çalıştırın."

index.php: Ana yönetim paneli.
delete.php: Görev silme.
ekle.php: Yeni görev giriş formu.
update_status.php: Görev "tamamlandı" butonu.
baglan.php: Veritabanı köprüsü.
gorevler.sql: veritabanı -> "StatusHomework 0=Devam, 1=Bitti".
